# String Methods

<br>

## includes

문자열이 들어있는지 검사

```javascript
"abcdefg".includes("c"); // true
```

## repeat

문자열을 반복

```javascript
"a".repeat(3); // "aaa"
```

## startsWith, endsWith

처음, 뒤의 문자열이 일치하는지 검사

```javascript
const string = "Hi, Nice to meet you";

string.startsWith("Hi");    // true
string.endsWith("you");    // true
```
